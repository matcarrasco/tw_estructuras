# tw_estructuras
Proyecto de estructuras

Guía para traerse el proyecto de python

https://gitforwindows.org/

1. Descarga git para windows y crea tu branch
2. git clone -b feature-alex https://github.com/matcarrasco/tw_estructuras.git
3. git pull origin master

Agregar cambios a tu branch
4. git add --all
5. git commit -m "Mensaje del commit"
6. git push
